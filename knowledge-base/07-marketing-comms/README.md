## Brand Overview

### Event Name
**Hack for RVA** - Richmond's First Civic Hackathon

### Tagline Options
- "Engineer the City We Want to Live In"
- "Build civic tech for Richmond. Win prizes. Ship real solutions."
- "7 Pillars. 48 Hours. Real Impact."

### Brand Voice
- **Inclusive:** Everyone is welcome, no experience required
- **Action-oriented:** Focus on building and shipping
- **Civic-minded:** About Richmond, for Richmond
- **Optimistic:** Solutions over problems

- **Primary:** Navy blue (#1E3A5F)
- **Accent:** Civic orange (#FF6B35)
- **Secondary:** Richmond gold (#FFD700)
