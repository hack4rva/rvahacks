# Leadership-to-Volunteer Mapping

## Overview

This document maps the 23 leadership roles (from `OrganizationalRoles.tsx`) to the volunteer and stakeholder roles they supervise. Each volunteer should know who they report to; each leader should know who they're responsible for.

## Escalation Paths

### Standard Escalation Hierarchy

```
Volunteer → Direct Report Leader → Event Manager → (911 if emergency)
```

### Common Escalation Scenarios

| Scenario | First Escalation | Second Escalation |
|----------|------------------|-------------------|
| Medical emergency | 911, then Event Manager | — |
| Code of Conduct violation | Event Manager | Legal/Compliance Lead |
| Technical platform issue | Platform Administrator | Event Manager |
| Venue/facilities issue | Site Captain or Science Museum Liaison | Event Manager |
| Team conflict | Volunteer Coordinator | Event Manager |
| Media inquiry | Social Media Manager | Event Manager |
| Judge issue | Judge Coordinator | Event Manager |
| Stakeholder concern | Respective coordinator | Event Manager |
